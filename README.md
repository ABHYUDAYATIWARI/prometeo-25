# Frontend setup
Procedure: 
- Install node
- Navigate to the cloned repository.
    ```
    cd <project_directory_name>     # prometeo-25
    ```
- Navigate to the frontend
    ```
    cd frontend
    ```
- Install all the  dependencies
    ```
    npm install
    ```
- Finally run frontend
    ```
    npm run dev
    ```
